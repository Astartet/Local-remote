Crea un nuevo repositorio en Github. Evita la creación de cualquier fichero: README, licencia o gitignore.
Abre Git Bash en tu equipo local. Te permitirá trabajar con git desde la línea de comandos.
Cambia al directorio de tu proyecto local. (cd)
Inicializa el directorio local como repositorio de git. (git init)
Añade los ficheros a tu nuevo repositorio local (git add)
Crea un index.html que contenga: nombre y apellidos del alumno
Usuario github del alumno
Estas instrucciones de la tarea
Crea un commit de los ficheros añadidos (git commit)
En Github, copia la url del repositorio remoto
En la línea de comandos de Git Bash, indica la url del repositorio remoto, para subir/sincronizar los cambios del repositorio local. (git remote add)
Confirma el cambio del repositorio remoto (git remote -v)
Sincroniza/sube los cambios del repositorio local a Github (git push)
